## 1) Main Post (Short + Strong)

Building NEXUS-ATMS has been one of the most exciting projects I have worked on so far.

It is an AI-driven Urban Traffic Management prototype that combines:
- RL-based signal control (DQN/PPO)
- LSTM traffic forecasting + anomaly detection
- Vision and IoT pipelines
- A live FastAPI dashboard + digital twin
- Safety and city modules (emergency corridor, carbon, cybersecurity, road maintenance, NL command)

Current benchmark snapshot:
- Average waiting time: 582.61s -> 9.65s (98.34% reduction)
- Average queue length: 26.36 -> 2.46 (90.66% reduction)
- Throughput: 496.8 -> 424.4 (still being optimized)

This is still an active build, and I am now focused on tighter real-world data wiring and production hardening.
