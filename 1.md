#### Understanding the RL Setup
```
STATE:  [queue_N, queue_S, queue_E, queue_W,        # vehicle counts per lane
         current_phase, phase_duration,              # what signal is doing now
         time_of_day, day_of_week]                   # context

ACTION: 0 = Give green to N-S
        1 = Give green to E-W
        2 = All-way pedestrian phase
        3 = Emergency hold

REWARD: -(total_queue_all_lanes) - (max_wait_time × 1.5)
        +50 if emergency vehicle clears intersection
```

#### Day 15–17: Build the SUMO-Gym Environment
```python
import gymnasium as gym
import traci
import numpy as np

class NexusIntersectionEnv(gym.Env):
    def __init__(self, junction_id):
        self.junction_id = junction_id
        self.action_space = gym.spaces.Discrete(4)
        self.observation_space = gym.spaces.Box(
            low=0, high=100, shape=(8,), dtype=np.float32)

    def step(self, action):
        self._apply_signal_phase(action)
        for _ in range(10):           # 10 simulation seconds per action
            traci.simulationStep()
        state = self._get_state()
        reward = self._compute_reward()
        return state, reward, False, False, {}

    def _compute_reward(self):
        total_wait = sum(
            traci.lane.getWaitingTime(lane)
            for lane in traci.trafficlight.getControlledLanes(self.junction_id))
        return -total_wait
```

```python
class IncidentDetector:
    def __init__(self):
        self.stopped_vehicles = {}    # tracks stationary vehicles
        self.bg_subtractor = cv2.createBackgroundSubtractorMOG2()

    def analyze_frame(self, frame, timestamp):
        incidents = []
        fg_mask = self.bg_subtractor.apply(frame)
        contours, _ = cv2.findContours(fg_mask, cv2.RETR_EXTERNAL,
                                        cv2.CHAIN_APPROX_SIMPLE)
        for cnt in contours:
            if cv2.contourArea(cnt) > 1000:
                x, y, w, h = cv2.boundingRect(cnt)
                vehicle_id = f"{x//50}_{y//50}"   # grid bucket ID

                    if stopped_for > 180:          # stopped > 3 minutes
                        incidents.append({
                            "type": "BREAKDOWN",
                            "location": (x, y),
                            "duration": stopped_for
                        })
                else:
                    self.stopped_vehicles[vehicle_id] = timestamp
        return incidents
```

```python
class SignalAnomalyDetector:
    def __init__(self):
        self.command_history = []

    def validate_command(self, command):
        # Rule 1: No junction can switch phase more than once per 10 seconds
        recent = [c for c in self.command_history
                  if c["junction"] == command["junction"]
                  and (command["time"] - c["time"]) < 10]
        if len(recent) > 1:
            return False, "RATE_LIMIT_BREACH — possible cyber attack"

        # Rule 2: All-green simultaneously is physically impossible
        if command.get("all_green_simultaneously"):
            return False, "IMPOSSIBLE_STATE — spoofing detected"

        # Rule 3: Commands must carry valid HMAC signature
        if not self._verify_signature(command):
            return False, "INVALID_SIGNATURE — unauthorized command"

        self.command_history.append(command)
        return True, "OK"
```

### What `VehicleDetector` implements

**3-tier fallback chain:**

| Priority | Backend | How it works |
|----------|---------|-------------|
| 1st | **YOLOv8n** | Loads `yolov8n.pt` via Ultralytics, runs on GPU (`cuda`) or CPU. Filters results to vehicle classes only. |
| 2nd | **OpenCV DNN** | MobileNet-SSD via `cv2.dnn`. Falls back here if `ultralytics` isn't installed. Needs weight files in `vision/weights/`. |
| 3rd | **Synthetic** | Generates random fake detections (0–12 vehicles per frame) for testing the pipeline without any model or camera. |

**YOLO specifically** ([lines 211–226](vision/detector.py#L211-L226)):
- Calls `self._model(frame, conf=self.conf, verbose=False)` — single-shot YOLOv8n inference
- Filters to **5 COCO vehicle classes only**: bicycle (1), car (2), motorbike (3), bus (5), truck (7)
- Extracts bounding box (`xyxy`), confidence, class ID, and center point
- Returns a list of `Detection` dataclass objects

**Supporting features:**
- FPS counter (rolling, updated every second)
- `draw()` method — annotates frames with bounding boxes, labels, confidence scores, and vehicle count
- Configurable confidence threshold (default 0.45)

### What it does NOT do
- No **tracking** — that's handled by tracker.py (IoU-based SORT)
- No **counting** — that's counter.py (polygon zone + line-cross)
- No **speed estimation** — that's speed_estimator.py
- No real camera feed — it processes whatever `np.ndarray` frame you pass in (SUMO screenshots, video files, or synthetic)
